## 電気関連
| 種類 | 最大数 |
| ---- | ---- |
| 発電系 | 合計64 |
| でんちゅう・むせんそうでんき 系 | 合計252 |
| 照明や自販機など | 511 (冷蔵庫系も含む。上限時は収納もできない)[^1] |
| CDプレイヤー | 32 |
| リフト乗り場 | 32 |
| エレベーター乗り場 | 32 |
| カメラ | 8 |
* イルミネーションライトは電力を消費しないが、カウントに含まれる。

## 植物
| 種類 | 最大数 |
| ---- | ---- |
| 畑(作物) | 511株, クラウド等は+64で576株 |
| きのみのなる木 | 226本 (頭突きもできなくなる) |
| 三角の木 | 251本(頭突きができなくなる) |
| 成長途中の木/花/作物 | 1024 (最大まで育つと別カウント) |
* 畑の上限注意は実のなる成長段階でのみ表示される

## 壊したブロックなどのアイテム
* 512個で、それを超すと最初にドロップしたものから消えていく。
* 落ちた作物やきのみはエリア移動で消えるが、壊したブロックは消えない。
* 貴重なものが画面外に落下した場合などは、ポケセン前に出現する。

## メタルなようこうろ
* 128個[^2]
* ようこうろ, コンクリートミキサーとの合計。
* クラフトだいは別カウント。

## クラフトだい・調理器具
* 合計128個
* まな板、フライパンなどの合計。

## タブレットなどに飾れる写真
* 10種類まで。同じ写真なら1種類にカウントされる。[^3]
* なお、アルバムから消しても、飾られている写真は消えない(置き換えたら二度と戻せないが)。

## キットハウス
* 40軒。
* ただし、立派な家のように家内でマップ切り替えが発生する家は2軒分にカウントされる。
* 建築中の状態は14個まで(建築中なら風車なども含む)。

## ラグ・ポスター
* それぞれ100枚


## 同時にとりひきが発生する数

ポケセンやフレンドリィショップを含んで、同時に1エリア合計10台までしか、レジが稼働しない。  
別のポケモンと取引したい場合は、ポケモンに2回話しかけ、「とりひきをやめる」でポケモンを入れ替える必要がある。

## 視界の上限

色が正常に表示されるのは５０マスまで
それ以上はだんだん薄くなり７８マスで単色になる
１７０マスから半透明になり１７５マスで見えなくなる
